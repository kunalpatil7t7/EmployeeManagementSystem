# Employee Management System – Spring Boot Backend

## Overview
A Spring Boot backend application for managing employee records, demonstrating **RESTful APIs, database integration, and role-based access**.  
The project follows a **layered architecture** (Controller → Service → Repository) for scalable and maintainable code.

---

## Key Features
- **CRUD Operations:** Create, Read, Update, Delete employees via REST API  
- **Role-Based Access:** Admin and User roles with Spring Security  
- **Database Integration:** MySQL with JPA/Hibernate  
- **Validation:** Ensures correct and consistent employee data  
- **Layered Architecture:** Controller, Service, Repository for clean separation of concerns  

---

## Technologies
- Java  
- Spring Boot  
- Spring Data JPA / Hibernate  
- MySQL  
- Maven
